# ⚡ Vidyut – Smart Digital Notebook

Vidyut is a simple web-based Smart Digital Notebook that allows users to create, organize, and manage notes digitally.  
It helps students keep their notes, subjects, and plans organized in one place.

---

## 🚀 Features

- 🔐 Simple Login System
- 📝 Create and Save Text Notes
- ✍ Handwritten Notes using Canvas
- 📚 Organize Notes by Subjects / Folders
- 📅 Planner for daily tasks
- 🗑 Edit and Delete Notes
- 📱 Responsive design for mobile and desktop

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
- Canvas API
